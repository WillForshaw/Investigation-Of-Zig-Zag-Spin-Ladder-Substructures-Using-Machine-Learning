# Investigation-Of-Zig-Zag-Spin-Ladder-Substructures-Using-Machine-Learning
Simulations + ML for frustrated magnetism in zig-zag spin ladders. Python computes m(B) via a 4Ã—4 transfer-matrix Ising model, sweeps J3/J2 and field, exports a phase-diagram dataset (Gridmatrix.csv). An ML module featurises structures and trains a classifier to find zig-zag ladders from crystallographic/topology data (ToposPro/pymatgen + sklearn).
